# **医用画像処理工学**
###　**担当教員：健康データサイエンス学部　助教　内田　航**


このサイトは、**医療科学部 臨床工学科 4年生（前期）**を対象とした医用画像処理工学の資料を掲載しています。

来年の国家試験対策としての学習はある程度完了しているものとします。本講義では、**国家試験範囲の復習**に加え、**画像処理工学の理論的な背景や応用事例**を紹介し、理解を深めることを目的としています。</span>


## **授業資料の共有**
